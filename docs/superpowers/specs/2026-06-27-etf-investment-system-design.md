# ETF 场内基金智能投资决策系统 — 设计文档

## 项目概述

基于总资产额度的 ETF 场内基金投资决策系统，根据当前市场状态和标的特征，自动推荐最优交易策略并生成买卖信号。

## 技术栈

| 层级 | 技术 | 用途 |
|------|------|------|
| Web 框架 | Streamlit | 全栈 Web UI，单文件即可运行 |
| 数据获取 | AKShare | 免费开源金融数据接口，覆盖 ETF 净值/行情/估值 |
| 数据处理 | Pandas + NumPy | 时间序列处理与向量化计算 |
| 技术指标 | TA-Lib | MACD/RSI/ATR/布林带等指标计算 |
| 策略回测 | Backtrader | 多策略并行回测引擎 |
| 绩效报告 | QuantStats | Sharpe/MaxDD/年化收益报告 |
| 组合优化 | PyPortfolioOpt | 有效前沿、最大Sharpe比率 |
| 可视化 | Plotly | 交互式K线图、净值曲线、持仓饼图 |

## 投资哲学基石

1. **不预测市场，只做配置**（达利欧）— 系统输出比例建议，不预测涨跌
2. **成本即敌人**（博格）— 回测必含佣金+滑点，步长过小自动警告
3. **纪律战胜情绪**（博格+达利欧）— 规则驱动信号，标准答案不犹豫

## 策略体系

| 策略 | 逻辑 | 适用市场 | 年化预期 |
|------|------|---------|---------|
| 估值定投 | PE分位低时多买，高时少买 | 全市场 | 6%-10% |
| 网格交易 | 预设区间低买高卖，震荡收割 | 震荡市 | 7%-12% |
| 趋势跟随 | 均线金叉买入，死叉卖出 | 趋势市 | 8%-15% |
| 网格+定投混合 | 定投打底仓 + 网格做波段 | 震荡+长期 | 8%-12% |

## 风控体系

- PE分位 > 70% 抑制买入信号
- 单标的仓位上限 = 总资产 20%
- 预留 30% 资金应对极端行情
- 跌破网格下限 10% 触发清仓警示
- 回测自动计入佣金+滑点
- 步长过小自动警告

## 功能模块

1. **Dashboard** — 总资产、持仓、收益、风险指标一览
2. **策略中心** — 选择ETF → 四策略回测对比 → 推荐最优
3. **交易信号** — 当前买/卖/持有 + 具体金额/价格
4. **交易记录** — 历史交易 + 收益统计
5. **设置** — 参数调整、标的管理
